# Changelog

## 1.0.0-SNAPSHOT

- Built multi-module annotation-driven Excel framework skeleton.
- Added core mapping engine with conversion, validation, i18n, async export.
- Added pluggable adapters for poi, easyexcel, jxls.
- Added Spring Boot starter auto-configuration.
- Added sample-web for single/chunk/async/template/error-writeback scenarios.
- Added benchmark module and baseline runner.
- Added unit and integration tests.
- Added bilingual documentation and release metadata.
