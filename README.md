# AutoExcel

AutoExcel is an annotation-driven Excel import/export framework for Java.

## Modules

- autoexcel-annotation
- autoexcel-core
- autoexcel-converter
- autoexcel-validator
- autoexcel-event
- autoexcel-exception
- autoexcel-util
- autoexcel-adapter-poi
- autoexcel-adapter-easyexcel
- autoexcel-adapter-jxls
- autoexcel-spring-boot-starter
- autoexcel-sample-web
- autoexcel-benchmark

## Quick Start

Maven:

```xml
<dependency>
    <groupId>io.github.aprilger</groupId>
    <artifactId>autoexcel-spring-boot-starter</artifactId>
    <version>1.0.1</version>
</dependency>
```

Gradle:

```kotlin
implementation("org.shixuan:autoexcel-spring-boot-starter:1.0.0")
```
